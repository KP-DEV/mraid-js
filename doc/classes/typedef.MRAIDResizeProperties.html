MRAIDResizeProperties::allowOffscreen : Booleanproperty

Tells the container whether or not it should allow the resized creative to be drawn fully/partially offscreen

  • true (default): the container should not attempt to position the resized creative
  • false: the container should try to reposition the resized creative to always fit in the getMaxSize() area

MRAIDResizeProperties::customClosePosition : Stringproperty

Either top-left, top-right, center, bottom-left, bottom-right, top-center, or bottom-center indicates the origin of the container-supplied close event region relative to the resized creative. If not specified or not one of these options, will default to top-right.

MRAIDResizeProperties::height : Numberproperty

Height of creative in pixels

MRAIDResizeProperties::offsetX : Numberproperty

The horizontal delta from the banner's upper left-hand corner where the upper left - hand corner of the expanded region should be placed; positive integers for expand right; negative for left

MRAIDResizeProperties::offsetY : Numberproperty

The vertical delta from the banner's upper left-hand corner where the upper left - hand corner of the expanded region should be placed; positive integers for expand down; negative for up
